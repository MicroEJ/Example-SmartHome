# Overview
MicroEJ Java library API: smarthome.

## Usage
Add the following line to your `module.ivy` or your `ivy.xml`:
> `<dependency org="com.microej.demo" name="smarthome" rev="..."/>`

# Requirements
  - EDC-1.2 or higher
