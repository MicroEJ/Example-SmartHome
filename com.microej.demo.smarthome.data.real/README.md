# Overview
This project contains the providers for:
* Zwave thermostat
* Zwave door sensors
* Philips hue lights

## Usage
Add the following line to your `module.ivy` or your `ivy.xml`:
> `<dependency org="com.microej.demo.smarthome.data" name="real" rev="..."/>`

# Requirements
  - EDC-1.2 or higher
