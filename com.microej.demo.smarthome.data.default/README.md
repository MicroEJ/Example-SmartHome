# Overview
This project contains a fake implemantation of the data provider of Smarthome.

# Usage
Add the following line to your `module.ivy` or your `ivy.xml`:
> `<dependency org="com.microej.demo.smarthome" name="data.fake" rev="..."/>`

# Requirements
